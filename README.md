# Laminar Maven resources

Laminar's embeddable libraries are available here:

```
<repositories>
	<repository>
		<id>laminar-repo</id>
		<url>https://github.com/jmdisher/Laminar/raw/maven/repo</url>
		<releases>
			<enabled>true</enabled>
		</releases>
		<snapshots>
			<enabled>true</enabled>
		</snapshots>
	</repository>
</repositories>
```
